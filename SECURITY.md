# Security Policy

## Supported Versions

All versions supported at the moment 

## Reporting a Vulnerability

To report a security bug, please create a [GitHub issue](https://github.com/andy778/EMAX-Tuya/issues) in this repo with a description of the security bug, the steps to reproduce it, affected versions, and, if known, mitigations for the issue.

We will respond to your GitHub issue as soon as possible.
